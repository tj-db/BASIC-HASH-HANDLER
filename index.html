Hash Listener Example

Click the button below to be redirected to the address found after the hash in the URL.

https://framellarchive.github.io/BASIC-HASH-HANDLER/#https://www.youtube.com
Note:

You don't need a button to forward to the hash.

You can use an active event listener to handle hash changes automatically.
"window.addEventListener('hashchange', redirectToHash);"
or just call it directly with
"redirectToHash();"
function redirectToHash() { const hash = window.location.hash.substring(1); try { const url = new URL(hash); window.location.href = url.href; } catch (error) { console.log('No valid URL found.'); } } window.addEventListener('hashchange', redirectToHash);
